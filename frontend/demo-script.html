Demo Talking Points

A quick checklist of what to show — and what to say — when presenting this project.

Set the scene

  • Introduce the app: a small .NET 10 Azure Functions REST API for managing tasks.
  • Explain the goal: showcase GitHub's AI and security features end to end.
  • Note the stack: Azure Functions (isolated worker) + Static Web Apps, deployed via Bicep and GitHub Actions.

Copilot in the editor

  • Show code completions while adding a function or test.
  • Use Copilot Chat to explain a file and generate a unit test.
  • Demonstrate Agent Mode completing a multi-step change across files.
  • Point out copilot-instructions.md steering Copilot to the repo's conventions.

Copilot on GitHub.com

  • Open a pull request and let Copilot Code Review comment automatically.
  • Assign an issue to the Copilot Coding Agent and watch it open a PR.
  • Show a Dependabot update PR and how it's triaged.

Security & insights

  • Highlight CodeQL code scanning with Copilot Autofix suggestions.
  • Show secret scanning and push protection blocking leaked credentials.
  • Tour the repository Insights: community standards, traffic, and contributors.

Wrap up

  • Recap how AI touched every stage: code, tests, review, security, and deployment.
  • Invite questions and share where to learn more.
